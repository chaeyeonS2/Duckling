# Duckling
#### **덕클링: 덕후의 덕과 착 달라붙다의 cling을 결합한 단어**

덕후들이 착 달라붙어 편하게 소통하는 공간을 의미하며 뉴진스 팬들을 타겟으로 기획된 모바일 웹페이지

_"덕클링에선 ‘꾸미기’ 기능을 통해 본인의 아바타를 자신의 취향에 맞게 꾸미고, ‘AR 카메라’ 기능을 통해 모바일 카메라로 쉽게 자신의 아바타를 활용하여 사진 및 영상 콘텐츠를 만들 수 있습니다. 내가 꾸민 아바타가 생일 카페에 간 모습, 최애가 추천한 맛집에 간 모습 등을 찍을 수 있죠! 또한 해당 콘텐츠를 ‘둘러보기’를 통해 서비스 유저들과 함께 공유할 수 있고 트위터로 바로 콘텐츠를 업로드할 수도 있습니다."_
